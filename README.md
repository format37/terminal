## Langchain ReAct agent
including tools:
* Shell (bash)
* DuckDuckGo Search
* Retrieval  
![image](assets/reason-plus-action-react-framework.png)  
### Requirements
* Linux
* OpenAI api key
### Install repo
```
git clone https://github.com/format37/terminal.git
cd terminal
python3 -m pip install -r requirements.txt
```
### Run agent
Configure your agent in config.json  
Then run:
```
python3 agent.py
```
